# 2026 Agent Skill 最佳实践报告：设计树 (2026 Skill Best Practices Report: Design Tree)

## 问题

如何构建一个系统性的、搜索优先（search-first）的软件开发 Agent 技能栈？本设计树规划"搜索优先 vs 预训练知识"命题的思考-分析-总结-输出路径，最终产出一份面向 2026 年的工程实践指南。

## 范围

### 包含
- 搜索优先范式的理论框架与决策模型
- 2026 Q1 行业全景（平台/协议/设计模式）
- 技能架构的核心要素分析
- 最佳实践框架（铁律/反模式/成熟度模型）
- 三项目交叉验证（loom / mattpocock / gstack）

### 不包含
- 具体编程语言或框架的技能实现教程
- AI Agent 的通用架构设计（已有大量资料）
- 本项目（loom）的代码变更建议

## 假设
- "搜索优先"指 Agent 优先通过工具（WebSearch / Grep / MCP）获取实时信息，而非仅依赖模型预训练的静态知识
- 读者是技能开发者（编写 SKILL.md 的人），不是终端用户
- 2026 年行业数据来自公开可查证的来源

## 开放分支

1. **2.1-2.4 行业数据**：需通过 web search 获取最新数据，部分数据可能因来源时效性而需要多源交叉验证

## 决策节点

1. **4.1 铁律清单**：10 条铁律的具体内容需要在报告撰写过程中根据分析结果确定，可能增减

## 决策

- 采用"三项目交叉验证"作为报告的核心方法论，而非单一案例分析
- 报告本身以搜索优先方式撰写——每个论点通过工具搜索获得实证
